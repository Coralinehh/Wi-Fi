# Local Dataset Directory

Place datasets here before running experiments:

```text
BVP_22guesture_all/
NTU-Fi_HAR/
NTU-Fi-HumanID/
```

Dataset files are intentionally ignored by git.

## Download Links

NTU-Fi-HAR and NTU-Fi-HumanID:

```text
https://drive.google.com/drive/mobile/folders/1R0R8SlVbLI1iUFQCzh_mH90H_4CW2iwt?usp=sharing
```

Widar 3.0:

```text
https://tns.thss.tsinghua.edu.cn/widar3.0/
```

Put the prepared BVP 22-gesture `.mat` files in:

```text
BVP_22guesture_all/
```

For the two NTU-Fi CSI datasets, keep the original split folders:

```text
NTU-Fi_HAR/
  train_amp/
  test_amp/
NTU-Fi-HumanID/
  train_amp/
  test_amp/
```
